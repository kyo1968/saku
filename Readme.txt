1. 起動方法
saku-1.0-SNAPSHOT.jar または saku-1.0-SNAPSHOT.exe をダブルクリック。

2. コンパイル方法
mvn package -> target/saku-1.0-SNAPSHOT.jarができる。

3. アッセンブリ方法
mvn assembly:assembly -> target/saku-1.0-SNAPSHOT.zipができる。

4. 開発環境
Java SE7 (JDK)
Maven 3.1
Eclipse 4.3 (Kepler)
  - Maven Integration for Eclipse (推奨プラグイン)
  - EGit (推奨プラグイン)
